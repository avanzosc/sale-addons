===========================
Sale Service Multiple Price
===========================

.. |badge1| image:: https://img.shields.io/badge/licence-AGPL--3-blue.png
    :target: http://www.gnu.org/licenses/agpl-3.0-standalone.html
    :alt: License: AGPL-3

|badge1|

With this module you define a different price for services, the first unit will
be invoiced with the price unit and all others with this new price.

This will only be taken into account when the invoice is based on timesheets.
