========================================
Sale order event attendee no create sale
========================================

* If when confirming an assistant they already have a sale order and a sale
  order line defined, DO NOT CREATE a new order.
* When canceling an assistant, do not remove the information from the sales
  order, and from the sales order line.
* In assistants, always show the sales order, and the sales order line.
